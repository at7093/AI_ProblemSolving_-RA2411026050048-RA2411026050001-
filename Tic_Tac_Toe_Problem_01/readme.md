Problem Description :

Case Study: 
A gaming company wants to create an AI opponent for a web-based Tic-Tac-Toe game. Write 
a Python program to build a Tic-Tac-Toe game with the following feature: the user should be 
able to interact with the game via an interactive web interface (GUI). The computer (AI) should 
always make the best possible move for the user's input. 
Use the following methods: 
• Minimax Algorithm 
• Alpha-Beta Pruning 
Compare both methods based on: 
• Execution time (which one is faster) 
• Number of nodes explored (how many game states are checked)
